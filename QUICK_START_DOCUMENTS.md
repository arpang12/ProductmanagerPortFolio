## 5-Minute Setup Guide

### Step 1: Open Admin Panel
1. Navigate to your portfolio site
2. Click "Admin" in the header
3. Log in if needed

### Step 2: Edit a Case Study
1. Find the case study you want to add documents to
2. Click the "Edit" button
3. Scroll down to find the "Document" section

### Step 3: Enable Documents
1. Check the checkbox next to "Document" to enable the section
2. You'll see the document manager appear

### Step 4: Add Your First Document
1. Click the **"+ Add Document"** button
2. Fill in the form:
   - **Document Name**: Give it a descriptive name (e.g., "User Research Report")
   - **Document URL**: Paste the link to your document
3. Click **"Add Document"**

### Step 5: Add More Documents (Optional)
Repeat Step 4 for each additional document you want to add.

### Step 6: Save Changes
1. Scroll to the top of the page
2. Click the **"Save Changes"** button
3. Wait for the success message

### Step 7: View Your Work
1. Click **"Back to Dashboard"**
2. Navigate to the public case study page
3. Scroll to the "Project Documentation" section
4. See your documents beautifully displayed!

## â“ Troubleshooting

### Document not showing?
- Make sure the Document section is **enabled** (checkbox checked)
- Verify the URL is publicly accessible
- Save your changes after adding documents

### Wrong icon appearing?
- The system auto-detects based on URL
- Make sure your URL includes the file extension

### Can't edit document name?
- Click directly on the name text
- It will become editable
- Press Enter or click outside to save
