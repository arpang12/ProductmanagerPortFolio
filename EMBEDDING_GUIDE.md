# Embedding Guide for Case Studies

This guide explains how to embed Figma prototypes, YouTube videos, and Miro boards in your case studies.

## üé® Figma Prototypes

### Easy URL Input (Automatic Conversion):
1. Open your Figma file
2. Copy the URL from your browser (any Figma URL works!)
3. Paste it in the Figma section URL field
4. **We'll automatically convert it to embed format** ‚ú®

### Supported URL formats:
- Regular file URL: `https://www.figma.com/file/ABC123/My-Design`
- Design URL: `https://www.figma.com/design/ABC123/My-Design` (newer format)
- Prototype URL: `https://www.figma.com/proto/ABC123/My-Design`
- Embed URL: `https://www.figma.com/embed?embed_host=share&url=...`

### Auto-conversion feature:
- Paste any Figma URL and we'll detect if it needs conversion
- Click "Convert URL" when prompted for optimal embedding
- Or keep the original URL - both formats work!

### Tips:
- Make sure your Figma file has public viewing permissions
- The embed will show your prototype in interactive mode
- Users can navigate through your design flows

## üìπ YouTube Videos

### Supported URL formats:
- Regular YouTube URL: `https://www.youtube.com/watch?v=VIDEO_ID`
- YouTube share URL: `https://youtu.be/VIDEO_ID`

### Example URLs:
```
https://www.youtube.com/watch?v=dQw4w9WgXcQ
https://youtu.be/dQw4w9WgXcQ
```

### Tips:
- Both URL formats work automatically
- Videos are embedded with privacy-enhanced mode
- No need to get special embed codes

## üìã Miro Boards

### Easy URL Input (Automatic Conversion):
1. Open your Miro board
2. Copy the URL from your browser (any Miro board URL works!)
3. Paste it in the Miro section URL field
4. **We'll automatically convert it to embed format** ‚ú®

### Supported URL formats:
- Regular board URL: `https://miro.com/app/board/uXjVOcKGjZo=/`
- Live embed URL: `https://miro.com/app/live-embed/uXjVOcKGjZo=/?moveToViewport=...`

### Auto-conversion feature:
- Paste any Miro board URL and we'll detect if it needs conversion
- Click "Convert URL" when prompted for optimal embedding
- We automatically set a good viewport for your board

### Tips:
- Make sure your board has appropriate sharing permissions
- The embed shows a live, interactive version of your board
- Users can zoom and pan around your board

## üöÄ Troubleshooting

### Common Issues:

**Figma not loading:**
- Check if the file has public viewing permissions
- Ensure you're using the embed URL, not the regular file URL

**YouTube video not showing:**
- Verify the video is not private or restricted
- Check that the URL is complete and correct

**Miro board not interactive:**
- Make sure you're using the live embed URL
- Check board sharing permissions

### Getting Help:
- Just paste any URL - we'll help you convert it if needed!
- Look for the blue "Convert URL" prompt for automatic conversion
- Check the live preview to see if your embed is working
- Green checkmarks indicate URLs are ready to embed
