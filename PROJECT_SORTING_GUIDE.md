## ğŸ¯ How to Use

### For Visitors

**To sort projects:**
1. Click "Sort by" dropdown
2. Select sorting option
3. Projects reorder instantly

**To filter by tag:**
1. Click "Filter" dropdown
2. Select a tag
3. Only matching projects show

**To clear filter:**
1. Select "All Projects" from dropdown
2. Or click "Clear filter" button (if shown)

### For You (Portfolio Owner)

**To make tags appear:**
1. Go to Admin Panel
2. Edit case studies
3. Add tags to each project
4. Tags automatically appear in filter

**Best practices:**
- Use consistent tag names
- Keep tags relevant
- Don't use too many tags per project
- Common tags work better than unique ones
