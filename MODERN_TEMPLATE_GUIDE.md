## Color Palette

### Primary Colors
- **Background**: Linear gradient from #f5f7fa to #c3cfe2
- **Text**: #4a4e69 (dark gray-blue)
- **Headings**: #9a8c98 (muted purple)

### Accent Colors
- **Pink**: #ec4899 (border-pink-400)
- **Purple**: #a855f7 (border-purple-400)
- **Teal**: #14b8a6 (border-teal-400)

- **Light Cards**: rgba(255, 255, 255, 0.8) with backdrop blur

## Typography

### Font Family
- **Primary**: 'Poppins' - Modern, clean sans-serif
- **Weights**: 300 (light), 400 (regular), 600 (semibold), 700 (bold)

### Text Hierarchy
- **H1**: 4xl-6xl, bold, gradient text shadow
- **H2**: 3xl, bold, colored headings
- **H3**: xl, bold, section titles
- **Body**: lg, regular weight, readable line height

## Best Practices

### Content Guidelines
1. **Keep Headlines Concise**: Works best with 1-2 line headlines
2. **Use Quality Images**: High-resolution images enhance the aesthetic
3. **Limit Metrics**: 3 key metrics work best for visual balance
4. **Clear Process Steps**: 3-5 steps provide optimal flow

### Design Recommendations
1. **Consistent Imagery**: Use images with similar color tones
2. **Balanced Content**: Avoid overly long text blocks
3. **Strategic White Space**: Let the design breathe
4. **Color Harmony**: Stick to the established color palette
